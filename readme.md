# nodejs-scraper

Use javascript technologies to crawl and click buttons on websites with gui.

## Installation

* [Node.js](https://nodejs.org/en/)
* [pNpM](https://pnpm.io/)

```bash
  cp .env.example .env
  pnpm install
```

## Usage

```bash
  node app.js
```
