# nodejs-scraper

(2022) Use JavaScript technologies to crawl and click buttons on websites with GUI.

## Installation

* [Node.js](https://nodejs.org/en/).

```bash
  cp .env.example .env
  npm install --global yarn
  yarn install
```

## Usage

```bash
  yarn start
```
